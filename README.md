# iOS Makefile
The universal makefile for my iOS projects distributes IPAs in seconds.

## Features
* Git log as release notes
* goo.gl URL
* QRCode for URL
* Upload to SFTP via rsync
* Mailgun emails list
* local OTA server with Bonjour address
* Send iMessages to tester's iPhone
